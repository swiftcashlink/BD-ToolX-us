## üöÄ Deployment Guide

Follow these steps to deploy the suite to your production environment:

### 1. Initialize Repository
```bash
git init
git add .
git commit -m "Initial commit: BDToolX Neural Suite v2.9"
git branch -M main
```

### 2. Connect Remote & Push
```bash
git remote add origin https://github.com/yourusername/shortlink-manager.git
git push -u origin main
```

### 3. Build & Deploy (GitHub Pages)
The project includes a pre-configured GitHub Action in `.github/workflows/deploy.yml`.
Simply pushing to the `main` branch triggers an automated build and deployment to GitHub Pages.

To deploy manually:
```bash
npm run deploy
```
